## üìñ Overview
This is a **Server-Side Rendered (SSR)** personal portfolio website built using **Spring Boot 3 and Thymeleaf**. Unlike static HTML sites, this project leverages the power of Java backend to serve dynamic content.

The standout feature is the **Integrated AI Chatbot**, designed to interact with recruiters and visitors, providing real-time answers about my skills, projects, and experience using Generative AI capabilities.

## üöÄ How to Run Locally

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/pranjalpatel044/Portfolio-Website.git](https://github.com/pranjalpatel044/Portfolio-Website.git)
    ```
2.  **Navigate to Project Directory**
    ```bash
    cd Portfolio-Website
    ```
3.  **Configure API Keys (Optional)**
    * Open `application.properties`.
    * Add your AI API Key if required for the chatbot to function locally.
4.  **Run the Application**
    ```bash
    mvn spring-boot:run
    ```
5.  **Access the Website**
    * Open browser and go to: `http://localhost:8080`

## üß† Why Spring Boot for a Portfolio?
While many portfolios are static, I chose **Spring Boot** to demonstrate my backend expertise. This project serves as a live proof of concept for:

* Integrating 3rd Party APIs (AI Models) in a Java environment.
